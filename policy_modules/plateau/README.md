# Plateau Seed

Origin dataset: UNSAT-like long-stagnation regimes.

This module covers long-running states with weak novelty and persistent
unresolved structure. It prepares the solver for restart pressure, core-like
focus, or heavier diversification.

Encourages:

- persistent-core pressure
- restart-like diversification
- memory-informed escape

Suppresses:

- repeating failed local search patterns
- low-amplitude wandering on stale basins

Failure mode targeted: long plateaus and UNSAT-like regions.
